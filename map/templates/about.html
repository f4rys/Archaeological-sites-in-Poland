ABOUT

Interactive map of archaeological sites in Poland as provided by dane.gov.pl website. Coordinates are provided via WMS service. Information about every place is stored in sqlite3 database and loaded upon click on a specific point.

DATA USED

  • Archaeological sites in Poland provided via WMS service and CSV file
  • Data from dane.gov.pl used under the CC BY 4.0 licence
  • Maps provided by Google Maps API

LIBRARIES

  • Django
  • Leaflet.js
  • Leaflet.wms.js
  • Leaflet.GridLayer.GoogleMutant
  • Require.js
  • Bootstrap
